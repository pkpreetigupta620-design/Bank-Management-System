# 🏦 Bank Management System (C++)

A console-based Bank Management System developed using C++, designed to simulate basic banking operations with a focus on core programming concepts and logical problem-solving.

This project was built as a learning exercise and is suitable for showcasing fundamentals during interviews, including at organizations like Deloitte.

## 📌 Project Overview
The application allows a user to securely manage a bank account using a PIN-based authentication system.  
It supports essential banking functionalities such as deposits, withdrawals, balance inquiry, and transaction history tracking.

## 🎯 Key Objectives
- Apply C++ fundamentals to a real-world scenario  
- Understand menu-driven program design  
- Implement basic security using PIN verification  
- Improve logical thinking and structured coding  

## 🛠 Technologies & Concepts Used
- Language: C++  
- IDE: Visual Studio Code  
- Core Concepts:  
  - Variables & Data Types  
  - Functions  
  - Loops  
  - Conditional Statements  
  - Switch Case  
  - Arrays (for transaction history)

## ⚙️ Features
- Account creation with 4-digit PIN  
- Secure deposit and withdrawal operations  
- Balance inquiry with authentication  
- Transaction history (last 5 transactions)  
- Menu-driven and user-friendly console interface  

## 🧠 Application Flow
1. User creates a bank account and sets a PIN  
2. System displays a menu of banking options  
3. PIN is verified before sensitive operations  
4. Selected operation is executed  
5. Transaction details are recorded  
6. Program runs until the user exits
